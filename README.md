# html-Birthday Invite Udemy (In Progress)

Overview:

This project is an in-progress introductory portfolio website created as part of a structured Udemy course. The goal of the project is to practice foundational HTML concepts by building a simple, static website that presents basic personal information and navigation elements.
The site focuses on clean structure, semantic markup, and linking content using standard HTML elements.

Technologies Used:

HTML

Concepts Demonstrated:

Semantic HTML structure
Anchor tags for internal and external navigation
Image embedding and basic layout
Organizing content into logical sections
Incremental development through coursework

How to Run:

Clone the repository
Open the index.html file in a web browser

Project Status:

This project is currently in progress and is being expanded as additional concepts are introduced in the course.

Lessons Learned:

Improved understanding of HTML document structure
Gained experience linking pages and assets correctly
Reinforced best practices for readable, organized markup

Future Improvements:

Add CSS for layout and styling
Improve accessibility and semantic structure
Expand content and navigation as coursework continues
